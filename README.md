# Apache HBase (apache-hbase)
Apache HBase is an open-source, distributed, versioned, non-relational database modeled after Google's Bigtable. It provides random, real-time read/write access to big data and runs on top of Apache Hadoop HDFS.

## APIs

### Apache HBase REST API
HBase provides a REST API (Stargate) for table operations, scans, and cell-level access, a Thrift API for cross-language access, and a comprehensive Java client API for all data operations including filters, coprocessors, and admin functions.
